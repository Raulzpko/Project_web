Important HTML tags

This page is used as a reference for the most common tags for HTML. In the process of making this i'll introduce how to make HTML tables via table and related tags.

The tables below don't include all the HTML tags but they do include many of the most important ones.

Block of elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1 – 6)
p Paragraph Include a paragraph of text
table Table Include a table
tr Table Row Add a table row of data
th Table Header Make a table header
td Table Data Add a table data cell
ol Ordered list Add an ordered list
ul Unordered List Add an unordered list

Inline Elements

Tag Name Purpose Example Result
em Emphatized Make emphatized text <em>The Covenant</em> The Covenant
strong Strong Make strong text <strong>The Covenant</strong> The Covenant
code Code Make monospace text <code>The Covenant</code> The Covenant
a Anchor Inlclude a link <a href="https://www.xkcd.com/"<Link here</a> Link here
img Image Include an image <img src="path/to/image" alt="Image description">
li List element Add a list element <li>Test of the list element</li> No example
